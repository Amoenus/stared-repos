Number Guessing Game

Here are some of the rules and ways to play the game.

  • 1. There are two players in this game, one is the user/player and other is the computer.
  • 2. Both user and computer will guess a number between 0-9
  • 3. The one who guesses early will be the the winner.
  • 4. If you want you can reset the game as and when you like.
  • Happy Guessing

So to begin the game just press the start button below.
